# 資産管理アプリ：AssetMaster JP 開発定義書

## 1. プロジェクト概要
日本株、米国株、投資信託、銀行預金、株主優待を統合管理する個人向け資産管理Webアプリ。
「今いくら持っているか」をリアルタイム株価と手動残高更新で可視化する。

## 2. 技術スタック
- **Frontend**: Next.js (App Router), Tailwind CSS
- **UI Component**: shadcn/ui, Lucide React (Icons)
- **Charts**: Recharts (資産推移・配分表示用)
- **Database/Auth**: Supabase (PostgreSQL)
- **State Management**: TanStack Query (React Query)
- **Stock API**: Yahoo Finance API (yfinance) または Alpha Vantage

## 3. 画面構成 (Pages)
- `/dashboard`: 総合サマリー（総資産、前日比）、資産構成円グラフ、推移グラフ、直近の優待アラート。
- `/assets`: 【統合管理画面】上部に資産登録/編集フォーム、下部に保有資産の一覧（カテゴリ別）を配置。
- `/analysis`: 詳細分析（セクター別比率、騰落率ランキング）。
- `/settings`: ユーザー設定、為替レート(USD/JPY)設定、CSVインポート設定。

## 4. 機能要件詳細
### 4.1 資産マスター管理 (統合画面)
- **銀行・現金**: 手動で現在の残高を上書き。前回との差分を自動で収支ログとして記録。
- **株式 (日米)**: 銘柄コード/ティッカー、保有数、取得単価を入力。
- **投資信託**: 銘柄名、保有口数、基準価額（取得時）を入力。
- **インライン編集**: 一覧リストから直接「残高」や「保有数」を更新可能にする。

### 4.2 株価・為替同期
- 日本株、米国株、投信の現在値をAPI経由で取得。15分間キャッシュ。
- 米国株資産は最新の為替レートでリアルタイム円換算。

### 4.3 株主優待管理
- 銘柄ごとに「優待内容」と「換算金額」を登録。
- 権利確定月（3月, 9月等）を保持し、当月が確定月の銘柄をハイライト。
- 年間の見込み優待総額を算出し、資産の一部として表示切替可能にする。

## 5. UI/UX デザイン指針
- **テーマ**: モダンフィンテック（ダークモード推奨）。
- **日本語最適化**: 金額は3桁カンマ区切り (`1,234,567円`)、日付は `YYYY/MM/DD` 形式。
- **カラー**: 上昇/プラスは `emerald-500`、下落/マイナスは `rose-500`。
- **モバイル対応**: 外出先での残高更新を考慮したレスポンシブ設計。

## 6. セキュリティ・データ保護
- 銀行のログインID/パスワードは一切保持しない。
- 証券口座等の認証情報も保持せず、銘柄コードと数量のみを管理する。
